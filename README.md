# BlackParrot RISC-V Simulation using Verilator

## Overview
This project demonstrates the simulation of the BlackParrot RISC-V processor using Verilator 5 on Ubuntu Linux.

## Objective
To successfully build and simulate an open-source RISC-V processor and verify its functionality.

## Tools Used
- Verilator 5
- Ubuntu Linux
- BlackParrot Processor

## Steps Performed
1. Cloned BlackParrot repository
2. Installed Verilator
3. Fixed build and lint errors
4. Configured simulation environment
5. Ran simulation successfully

## Output
- Simulation executed successfully
- PASS status observed
- Processor modules terminated correctly

## Key Learning
- Understanding of RISC-V architecture
- Verilog simulation using Verilator
- Debugging hardware design errors
